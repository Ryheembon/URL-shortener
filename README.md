### Backend Setup

   ```bash
   cd backend
   ```

2. Run the Go server:
   ```bash
   go run main.go
   ```
   The server will start on `http://localhost:8080`

### Frontend Setup

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```
   The frontend will be available at `http://localhost:3000`
